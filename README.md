# Calculator
This is a regular 4-function calculator made using WinUI3
Run this app through Visual Studio after downloading the WinUI 3 packages:
https://learn.microsoft.com/en-us/windows/apps/get-started/start-here?tabs=wingetconfig
In Solution Explorer, right-click Calculator.slnx and choose "Set as Startup Project."
The project name should turn bold when it's set correctly
(Or just double click Calculator.slnx after cloning the repository. It should set up the project after that)
